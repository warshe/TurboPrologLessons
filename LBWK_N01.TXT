      1. Базы данных в среде Микро Пролога и Турбо Пролога

     Цель лабораторной работы:
 - на примере конкретных программ ознакомится с основными  меха-
   низмами Пролог-системы,
 - отработать понятия: атом, переменная, цель, успех и неуспех в
   достижении цели, факты, правила, рекурсивное определение пра-
   вил, вопросы, унификация переменных,
 - научиться  составлять  простейшие базы данных в среде Микро и
   Турбо Пролога,
 - научиться  изображать диаграммы вывода для конкретных целей в
   программе.

1.1. Введите в Микро Прологе программу 1 (exmpl101.p):

        ((рыба окунь))
        ((рыба щука))

     Введите несколько запросов,  отвечающих на вопрос, знает ли
     Пролог-система данную рыбу.  Например,
                   (рыба окунь), (рыба лещ).
     Как отреагирует Пролог-система на  конструкцию  (рыба Y)?

1.2. Введите программу 2 (exmpl102.p):

        ((рыбы)(рыба X)(PP X)(PP щука лещ))
        ((рыба окунь))

     Отработайте цели (рыба окунь),  (рыба лещ), (рыбы). Проана-
     лизируйте работу Пролог-системы.

* 1.3.  Написать  программу,  такую,  что в ответ на конструкцию
     (СЕЗАМ) Пролог-система  напечатает  текст  этой  программы.
     Программа содержит два предложения,  второе предложение со-
     держит текст первого предложения, а информация об остальном
     тексте  второго  предложения  должна  содержаться в первом.
     Программа похожа на приведенную программу 2.

* 1.4.  В среде Турбо Пролога создайте базу данных по  выбранной
     теме, содержащей по 4 - 5 унарных, бинарных, тернарных,...,
     отношений.  Дополните ее некоторым числом правил, среди ко-
     торых  есть и рекурсивные.  Общее число предложений порядка
     20.  Проведите тестирование программы, вводя различные воп-
     росы (цели).

                 Темы заданий (выдаются преподавателем):

 1) любимые виды спорта,             14) геометрические фигуры
 2) покупки,                         15) функции,
 3) интересы,                        16) множества,
 4) автомобили,                      17) четырехмерный куб и его
                                         грани,
 5) книги,                           18) матрицы,
 6) телефоны и адреса,               19) числа,
 7) друзья и знакомые,               20) деревья и графы,
 8) рыбы,                            21) языки программирования,
 9) птицы,                           22) расписание занятий,
10) животные,                        23) учебный план,
11) химические элементы,             24) учебные занятия,
12) международная система единиц,    25) электрическая схема,
13) семейные отношения,              26) свободная тема.

       2. Рекурсивное вычисление функций на Микро Прологе

2.1. Проанализируйте программу exmpl103.p, вычисляющую факториал
     натурального числа.  Здесь PP, SUM, TIMES - встроенные пре-
     дикаты, определенные следующим образом:
        (PP X) - предикат печати X, всегда принимающий значение
                 'истинно',
        (SUM X Y  Z)  -  предикат  суммы,  принимающий  значение
                 'истинно',  если  справедливо равенство
                 X + Y = Z,  и вычисляющий один  из  аргументов,
                 если заданы другие два,
        (TIMES X Y Z) - предикат произведения,  принимающий зна-
                 чение 'истинно', если справедливо равенство
                 X * Y = Z,  и вычисляющий один  из  аргументов,
                 если заданы другие два (арифметические операции
                 выполняются в поле целых чисел).
     Вычислите несколько значений n!, экспериментально определи-
     те "длину" полки целей.  Что произойдет при вычислении 0! ?
     Исправьте программу таким образом, чтобы было возможным вы-
     числение 0!.

* 2.2.  Напишите на  Микро Прологе  программу,  которая  получив
     конструкцию (Fib X), вычисляет и печатает число Фибоначчи с
     номером X.  Особо одаренные могут написать  программу,  вы-
     числяющую обобщенные числа Фибоначчи.

2.3. Конструкция (x|X) разделяет список (a1 a2 ... an) на голову
     x=a1 и хвост X=(a2  ...  an).  Разберите  работу  следующей
     программы (exmpl104.p):
        ((N (x|X) 1) (PP x))
        ((N (x|X) Y) (SUM Z 1 Y) (N X Z))
     подав на вход цели (N (a b c d e) k) для k = 0,  1, ..., 6.
     Для чего предназначена программа? Что происходит в случае
     k = 6 ?

* 2.4. Напишите программу на Микро Прологе, подсчитывающую число
     элементов списка.

2.5. Разберите следующую программу (exmpl105.p):
        ((CAT X Y) (Cat X Y Z) (PP Z))
        ((Cat (x|X) Y (x|Z)) (Cat X Y Z))
        ((Cat () Y Y))
     Подайте на вход цель (CAT (A B) (C D)) и ей аналогичные.

* 2.6. Напишите программу на Микро Прологе, реверсирующую (обра-
     щающую) список ( (a1 ... an) --> (an ... a1) ).

* 2.7. Числа Каталана C(x) определяются рекурсивно:
     C(1) = 1, C(2) = 1, C(n+1) = C(1)*C(n) + ... + C(n)*C(1).
     Напишите программу на Микро Прологе,  вычисляющую по задан-
     ному числу x число C(x).

* 2.8. Индивидуальное задание (выдается преподавателем).

   1. Напишите программу  сложения  двух  неотрицательных  целых
чисел, используя аксиоматику: 0+N=N, (N)+1=(N+1).
   2. Напишите программу умножения  двух  неотрицательных  целых
чисел как повторяющееся сложение. (Операция сложения известна.)
   3. Напишите программу возведения натурального числа  в  целую
неотрицательную  степень как повторяющееся умножение.  (Операция
умножения известна.)
   4. Напишите программу, определяющую отношение меньше или рав-
но (меньше, больше или равно, больше) на множестве целых неотри-
цательных чисел. Используйте ее для написания программы минимума
(максимума) двух таких чисел.
   5. Напишите программу,  определяющую четность (нечетность це-
лого числа (допустимая операция: сложение или вычитание).
   6. Напишите программу целочисленного деления (допустимая опе-
рация: сложение или вычитание).
   7. Напишите программу,  определяющую отстаток от деления двух
натуральных чисел.
   8. Напишите   программу,   вычисляющую   функцию   Аккермана:
      A(0,N)=N+1, A(M,0)=A(M-1,1), A(M,N)=A(M-1,A(M,N-1)).
   9. Напишите программу,  вычисляющую наибольший общий делитель
двух натуральных чисел по алгоритму Евклида.
  10. Напишите программу,  определяющую наличие цепи между двумя
вершинами графа.
  11. Напишите программу,  определяющую, что два элемента списка
являются соседними.
  12. Напишите программу, определяющую последний элемент списка.
  13. Напишите программу, удваивающую список: каждый элемент но-
вого списка встречается последовательно два раза.
  14. Напишите  программу,  удаляющую все вхождения данного эле-
мента из списка.
  15. Напишите программу, находящую наибольший элемент списка.
  16. Напишите программу, определяющую сумму элементов списка.
  17. Напишите  программу  удаления  всех  повторных   элементов
списка.
  18. Напишите  программу,  меняющую  в списке префикс и суффикс
местами.
  19. Напишите программу определения  четности  или  нечетности
перестановки.
  20. Напишите программу,  определяющую,  является ли данная пе-
рестановка циклом.

* Задания, отмеченные знаком *, сдаются в виде законченных прог-
  рамм.
